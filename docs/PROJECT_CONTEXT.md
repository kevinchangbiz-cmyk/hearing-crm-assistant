# PROJECT_CONTEXT — Hearing CRM Assistant

## 專案背景
本系統由台灣助聽器門市驗配師使用。目標是減少 CRM 服務紀錄、提醒事項與行政輸入時間。

核心目標：**服務完成 → 語音輸入 → AI 整理 → 複製 CRM，30 秒內完成。**

## 真正要解決的問題
- CRM 服務紀錄耗時
- Google Calendar 容易漏登
- 行政事項容易忘記

## 不是要做的東西
- ERP
- CRM 系統
- KPI 分析
- 客流分析
- 多 Agent 平台

## 設計原則
1. 越簡單越好
2. 減少按鈕
3. 減少輸入
4. 以語音為主
5. 配合驗配師既有習慣

## 開發優先順序
- P1 CRM 服務紀錄
- P2 Google Calendar 提醒
- P3 Google Sheet 備份
- P4 UI 優化

## 一句話定義
> 協助助聽器驗配師快速完成服務紀錄的小工具。
