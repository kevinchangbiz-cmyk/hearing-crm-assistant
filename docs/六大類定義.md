# 六大類定義

> 大致相符即可，畢竟很多內容會重疊。

1. **保養**：助聽器保養、更換耳管、耳塞、耗材、清潔、除濕；如有檢耳也可輸入在此，或者輸入在 `<調整>`。
2. **調整**：客戶反應的問題，根據問題做出處置。可能是助聽器聲音調整、耳塞耳模不舒服調整、聽力下降重新檢查調整、也可能是根本沒戴好進行衛教或教學，較複雜難以細說。
3. **維修**：故障原因、時間多久、檢查後發現的情形、如何處置。
4. **聽檢**：新客 (第一次配助聽器) 聽力檢查和選配要選這個；舊客例行性聽力檢查。
5. **總公司客服**：為總公司客服部紀錄使用，門市很少用到。
6. **其他**：雜項、備註事項、注意事項。
